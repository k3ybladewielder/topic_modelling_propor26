# Topic Modelling Experimentation Pipeline
This repo contains the code of the Experimentation Pipeline used in the paper **Experimental Evaluation of Topic Modeling Methods for Categorizing Irregularities in Health-related news** using the [Health Related News Dataset](https://zenodo.org/records/18039964) presented at the The 17th International Conference on Computational Processing of Portuguese will be held at Salvador, BA.
